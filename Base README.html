Direction to complete README.html for a project. 1) Start with a copy of this file. Place it in projects/PROJECT_NAME. 2) Rename it to README.html. 3) Copy the README.md contents from the project you are adding/editing. 4) Go to this web site: https://markdowntohtml.com/ 5) Paste the README.md contents on the left-hand side of that side, in Enter Markdown. 6) You'll see the content on the right-hand side. Switch to Raw HTML and copy it all. 7) Go back the new README.html. Edit the file, paste the Raw HTML over this set of directions.
